**Conditional Formatting**

-Select Cells
-Go to Home Tab > Styles > Conditional Formatting
-Formatting Rule> Add new
-Select Rule > Cells that contain
-Edit Rule Description
-Go to Format
-Do the Formatting
-Add desired no of rules

Select cells
-Go to Home Tab > Styles > Conditional Formatting(DropDown)
select Data Bars > Select Any
*it shows bar in percent, where the 100% value is the highest value in the selected cells.
-------------------------------------------------------------------------------------------

Select cells
-Go to Home Tab > Styles > Conditional Formatting(DropDown)
select Icon Set > Select Any
*it shows Range color in the selected cells.
**RANGE = MAX - MIN**
RANGE / No of Icons (if 3 -> 33%)
icon1 if Value 0-33%
icon2 if Value 33-66%
icon3 if Value 66-100%
